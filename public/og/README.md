# OG images

`default.svg` is the fallback social-share card (1200×630) referenced by every page that doesn't override `ogImage`.

For richer social previews on Twitter/X and LinkedIn, export a 1200×630 PNG version of this SVG (or design fresh per-page cards) and save them here as e.g. `og/part-1.png`, then pass `ogImage="/og/part-1.png"` to `<Base>` on that page.

Three custom share images for launch (per the launch kit) will live here:

- `launch-1.png` — handbook + DOI
- `launch-2.png` — quiz teaser
- `launch-3.png` — translator teaser
